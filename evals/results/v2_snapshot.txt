================================================================================
  TOOLS  (7 total — 1 added)
================================================================================

  1. list_customers(region, industry, crm_stage, product, limit=100)  [NEW]
     Enumerates customers filtered by optional partial-match (LIKE) fields.
     Joins customers → scenarios → products to expose primary_product name.
     Returns: customer_id, name, industry, subindustry, region, crm_stage,
              account_health, scenario_id, primary_product.
     Use for: any question scoped to a region, segment, CRM stage, or product.
     SQL: customers JOIN scenarios ON scenario_id JOIN products ON primary_product_id
          with optional WHERE clauses using LIKE '%...%' on each filter param.

  2. search_artifacts(query, limit=10)  [unchanged]
     Full-text keyword search across artifact titles, summaries, and content.
     Returns: flat list of (id, type, title, summary, customer_id, scenario_id).

  3. find_pattern_across_customers(query, limit=50)  [unchanged]
     FTS search grouped by customer — for "widespread or one-off?" questions.
     Returns: customers_affected count + artifact titles per customer.

  4. customer_artifacts(name_or_id)  [unchanged]
     Returns all artifacts for a known customer (by name or customer_id).

  5. scenario_summary_tool(scenario_id)  [unchanged]
     Full context for a scenario in one shot (scenario, customer, impl, artifacts).

  6. artifact_full_text(artifact_id)  [unchanged]
     Fetches full content_text of a single artifact.

  7. sql_query(sql)  [unchanged]
     Read-only SELECT against the SQLite database.
